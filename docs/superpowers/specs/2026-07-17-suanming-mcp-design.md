# 算命 MCP Server — 设计文档

## 概述

一个传统中国风 + 程序员整活风的算命 MCP Server，发布为 npm 包，供 Claude Code、OpenCode 及任何 MCP 兼容工具使用。每次算命结果渲染为水墨风格 HTML 页面，浏览器打开查看。

## 架构

```
suanming-mcp    (npm 包名)
├── src/
│   ├── index.ts              # MCP Server 入口，注册 6 个 tool
│   ├── tools/
│   │   ├── bazi.ts           # 八字排盘
│   │   ├── liuyao.ts         # 六爻起卦
│   │   ├── qian.ts           # 抽签
│   │   ├── name.ts           # 姓名打分
│   │   ├── mingming.ts       # 起名
│   │   └── coder.ts          # 程序员黄历
│   └── render/
│       └── template.ts       # 水墨 HTML 模板引擎
├── package.json
└── README.md
```

**零外部依赖**，纯 TypeScript + MCP SDK，发布后 `npx suanming-mcp` 直接注册。

## 六个 Tool

### 1. `fortune_bazi` — 八字排盘

- **输入**：出生年月日时（公历或农历）+ 性别
- **输出**：四柱八字、五行分布、日主分析、今日运势简评
- **算法**：公历转农历 + 天干地支纪年法推算

### 2. `fortune_liuyao` — 六爻起卦

- **输入**：用户心中默念的问题（可选）
- **输出**：本卦 + 变卦、变爻位置、卦象解读
- **算法**：时间戳种子随机生成 6 爻，老阴老阳为变爻，内置 64 卦释义

### 3. `fortune_qian` — 抽签

- **输入**：无（或用问题做种子）
- **输出**：签号（1-100）、签文（四句七言诗）、解签白话文、吉凶标签
- **算法**：100 首签文预置，按种子随机抽取

### 4. `fortune_name` — 姓名打分

- **输入**：姓名（2-4 字）
- **输出**：五格数理（天格/人格/地格/外格/总格）、三才配置、各格吉凶、综合评分
- **算法**：标准繁体笔画数 + 五格剖象法公式

### 5. `fortune_mingming` — 起名

- **输入**：姓氏（必填）、出生年月日时（可选）、性别（可选）、字数偏好（默认 3 字）
- **输出**：5 个候选名字，每个附带五格分数、八字五行分析、寓意解读、推荐指数
- **算法**：内置 500-1000 字起名用字库按五行分类，结合八字补缺生成候选，五格筛选

### 6. `fortune_coder` — 程序员黄历

- **输入**：编程语言
- **输出**：今日宜忌（幽默向）、代码气运指数（bug 率/效率/摸鱼）、幸运语言/背运语言、一句话忠告
- **算法**：日期 + 语言名做种子，全随机但同天同语言结果一致

## 水墨 HTML 视觉设计

### 配色

| 用途 | 颜色 | 说明 |
|------|------|------|
| 背景 | `#f5f0e8` 宣纸色 | 带纹理噪点 |
| 主文字 | `#2c2416` 浓墨 | 深褐黑 |
| 标题 | `#1a1a1a` 重墨 | 纯黑偏暖 |
| 点缀 | `#c41e1e` 朱砂红 | 印章、标签 |
| 辅助线 | `#8b7355` 淡墨 | 分隔线、边框 |
| 印章底 | `#d4342c` 印泥红 | 圆角方形章 |

### 字体

- 标题：Google Fonts `Ma Shan Zheng`（马山正毛笔体）
- 正文：`Noto Serif SC` / `SimSun` / `STSong` 宋体系列

### 视觉元素

- 宣纸纹理：CSS 多层噪点叠加
- 墨迹晕染：`box-shadow` + `border-radius` 不规则边缘
- 印章：CSS 圆角方形 + 旋转 15°，刻"玄机阁"三字
- 八卦符号：Unicode 原生 ☰☱☲☳☴☵☶☷
- 淡入动画：内容从墨色渐显

### 各 Tool 模板变体

| Tool | 页面差异 |
|------|---------|
| 八字 | 四柱表格 + 五行圆环 |
| 六爻 | 本卦变卦符号 + 爻位标注 |
| 抽签 | 签筒意象 + 签文竖排 |
| 姓名打分 | 五格卡片 + 评分数字 |
| 起名 | 候选名字列表 + 印章推荐 |
| 程序员黄历 | 水墨混搭像素风小元素 |

## 技术选型

- **语言**：TypeScript
- **MCP SDK**：`@modelcontextprotocol/sdk`
- **HTML 生成**：模板字符串拼接，零依赖，内联 CSS
- **Google Fonts**：运行时加载 `Ma Shan Zheng`，无构建依赖
- **发布**：npm，包名 `suanming-mcp`

## 用户使用方式

### Claude Code 配置

```json
{
  "mcpServers": {
    "suanming": {
      "command": "npx",
      "args": ["-y", "suanming-mcp"]
    }
  }
}
```

### OpenCode 配置

```yaml
mcp_servers:
  suanming:
    command: npx
    args: ["-y", "suanming-mcp"]
```

然后直接对话"帮我算下八字"、"给我抽个签"、"今天适合写 Python 吗"即可触发。
